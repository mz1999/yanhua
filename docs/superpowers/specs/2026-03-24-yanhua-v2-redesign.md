# 言画视频创作系统 v2.0 - 设计文档

**日期**: 2026-03-24
**版本**: v2.0
**状态**: 待审核

---

## 1. 变更概述

### 1.1 核心变更

| 变更项 | v1.0 | v2.0 |
|--------|------|------|
| **流程** | 5步（提示词→图片→视频→配乐→发布） | 2步（图片→视频） |
| **配乐** | 手动上传配乐合并 | 即梦自动配乐 |
| **模板** | 5个预设模板 | 移除，完全自定义 |
| **文化元素** | 仅限中国（岭南、江南等） | 全球化（亚洲/欧洲/美洲等） |
| **模式** | 全自动+分步 | 仅分步模式 |
| **风格** | 基础 Tailwind | 极简高级设计 |

### 1.2 设计目标

- **极简主义**：大量留白，减少视觉噪音
- **高级感**：精致字体、微妙动效、专业质感
- **高效流程**：两步完成，快速迭代
- **全球化**：丰富的文化元素选择

## 3. 极简高级视觉设计

### 3.1 设计理念

**"Less but better"** - Dieter Rams

- 大量留白，让内容呼吸
- 每个元素都有存在的理由
- 微妙而非夸张
- 精致到像素级

### 3.2 色彩系统

```css
/* 基础色板 */
--color-bg: #fafafa;           /* 近乎纯白 */
--color-surface: #ffffff;       /* 纯白表面 */
--color-elevated: #ffffff;      /* 悬浮层 */

/* 文字色阶 */
--color-text-primary: #0a0a0a;  /* 标题 */
--color-text-secondary: #525252; /* 正文 */
--color-text-tertiary: #a3a3a3; /* 辅助 */

/* 强调色 - 极深蓝灰 */

/* 状态色 - 饱和度极低 */

--color-loading: #6366f1;

/* 边框 - 极淡 */
--color-border: #e5e5e5;
```

### 3.3 字体系统

**主字体**: `"SF Pro Display", "Inter", -apple-system, sans-serif`
**辅助字体**: `"SF Pro Text", system-ui, sans-serif`

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display | 48px | 300 | 1.1 | 首页标题 |
| H1 | 32px | 600 | 1.2 | 页面标题 |
| H2 | 24px | 600 | 1.3 | 区块标题 |
| H3 | 18px | 500 | 1.4 | 卡片标题 |
| Body | 16px | 400 | 1.6 | 正文 |
| Small | 14px | 400 | 1.5 | 辅助文字 |
| Caption | 12px | 500 | 1.4 | 标签（大写+字间距） |

### 3.6 动效规范

```css
/* 微交互 - 0.15s */
--duration-micro: 150ms;

/* 标准 - 0.2s */
--duration-standard: 200ms;

/* 复杂动画 - 0.3s */
--duration-complex: 300ms;

/* 入场动画 - 0.5s */
--duration-enter: 500ms;

/* 缓动函数 */
--ease-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
```

**关键动画**:
- 页面加载：淡入 + 轻微上移（opacity 0→1, translateY 20px→0）
- 卡片悬浮：上移 2px + 阴影增强
- 选中状态：边框色变化 + 轻微缩放 pulse
- 加载状态：优雅脉冲动画（非旋转）

## 6. 数据结构变更

### 6.1 简化后的 Task 模型

```prisma
model Task {
  id              String   @id @default(uuid())
  status          String   // 'generating_images' | 'selecting' | 'generating_video' | 'completed'

  // 配置输入
  style           String
  scene           String
  composition     String
  lighting        String
  culture         String   // 格式: "地域|具体元素"
  description     String?  // 补充描述

  // 输出
  images          String?  // JSON 数组，4张图片URL
  selectedImage   String?  // 选中的图片URL
  videoUrl        String?  // 生成的视频URL

  // 发布内容
  title           String?
  content         String?
  tags            String?  // JSON 数组

  // 元数据
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt
}
```

### 6.2 删除的字段

- `mode` - 不再需要（只保留分步模式）
- `templateId` - 不再需要模板
- `era`, `region` - 合并到 culture
- `prompt` - 前端不再展示
- `finalVideoPath` - 视频直接即梦生成
- `bgmPath` - 自动配乐，无需上传
- `errorMessage` - 简化错误处理

### 7.1 删除的 API
