# 言画配置体系 v3 重新设计方案

## 问题分析

当前配置体系的主要问题：
1. **维度边界不清**：赛博朋克、工业风、地中海等既有视觉特征，又有文化内涵，分类混乱
2. **视觉风格定义不明确**：治愈系是核心，但其他类别（电影感、极简等）与之不在同一层级
3. **空间类型穷举困难**：社区咖啡馆等复合空间难以枚举
4. **光影与其他维度冲突**：如选了"晴天"又与"阴天散射"冲突
5. **缺少地域维度**：无法表达"广州+昭和复古"这种组合

---

## 新配置体系（7个维度）

### 维度1：核心情绪（单选）- 最重要

**定位**：画面要传达的核心情感，决定了整体基调

| 选项 | 说明 | 适用场景 |
|-----|------|---------|
| **治愈系** | 温暖、柔和、让人心安 | 主打选项，日常舒适场景 |
| **宁静系** | 静谧、禅意、独处感 | 阅读、冥想、雨天 |
| **怀旧系** | 复古、时光感、故事性 | 老街、旧物、回忆 |
| **浪漫系** | 温柔、诗意、梦幻 | 黄昏、花海、约会 |
| **孤独系** | 清冷、疏离、思考感 | 深夜、窗边、独处 |

**设计理念**：这是整个画面的"情绪底色"，其他维度都围绕这个情绪展开。

---

### 维度2：建筑风格/文化基因（单选）

**定位**：空间的"建筑语言"和"文化元素"，决定"这是什么风格的空间"

#### 中国传统
- **岭南建筑**：满洲窗、花砖、骑楼、趟栊门
- **江南园林**：白墙黛瓦、漏窗、太湖石、曲径
- **北京胡同**：四合院、青砖灰瓦、枣树、鸟笼
- **上海石库门**：老钢窗、Art Deco、花园里弄
- **川西民居**：竹编、天井、茶馆、慢生活
- **云南民族**：扎染、火塘、梯田、多彩民族风
- **闽南建筑**：红砖厝、燕尾脊、出砖入石
- **徽派建筑**：马头墙、小青瓦、三雕艺术

#### 日式
- **京都町屋**：格子窗、土间、中庭、和纸
- **日式侘寂**：枯山水、苔藓、残缺美、禅意
- **昭和复古**：昭和风、City Pop、怀旧电器
- **冲绳琉球**：红瓦、狮子、亚热带海洋风
- **东京极简**：水泥、落地窗、都市冷峻

#### 东南亚
- **南洋复古**：花砖、藤编、百叶窗、娘惹文化
- **巴厘岛**：石雕、茅草、热带植物、泳池
- **泰北兰纳**：木雕、金箔、寺庙元素

#### 欧洲
- **地中海希腊**：蓝顶白墙、拱门、海洋元素
- **托斯卡纳**：红瓦、橄榄树、意式乡村、做旧墙面
- **南法乡村**：薰衣草、石屋、铁艺、田园野趣
- **北欧斯堪的纳维亚**：原木、hygge、极简功能美学
- **英式乡村**：壁炉、碎花、下午茶、庄园
- **巴洛克/洛可可**：华丽、繁复、宫廷感

#### 美洲
- **美式乡村**：手工质朴、Shaker风格、壁炉
- **加州Mid-Century**：棕榈树、泳池、复古现代
- **纽约Loft**：工业风、落地窗、都市精英
- **墨西哥**：色彩鲜艳、瓷砖、仙人掌、亡灵节

#### 其他
- **波西米亚**：自由奔放、民族元素、地毯织物
- **工业风**：金属、水泥、粗犷原始
- **包豪斯**：几何、功能主义、红黄蓝
- **赛博朋克**：霓虹、未来、科技感

**设计理念**：这是"建筑/空间的文化基因"，与"地域"维度可以叠加出有趣效果（如：岭南建筑+上海地域=上海老洋房里的岭南元素）。

### 维度3：地域（单选，可选）

**定位**：真实的地理位置，决定"在哪里"

| 地区 | 城市示例 |
|-----|---------|
| **华南** | 广州、深圳、香港、澳门、厦门 |
| **华东** | 上海、杭州、苏州、南京 |
| **华北** | 北京、天津、青岛 |
| **西南** | 成都、重庆、昆明、大理 |
| **西北** | 西安、敦煌、乌鲁木齐 |
| **华中** | 武汉、长沙 |
| **东北** | 哈尔滨、大连、沈阳 |
| **日本** | 东京、京都、大阪、冲绳、北海道 |
| **东南亚** | 新加坡、曼谷、清迈、巴厘岛、槟城 |
| **欧洲** | 巴黎、普罗旺斯、托斯卡纳、圣托里尼、伦敦 |
| **美洲** | 纽约、洛杉矶、旧金山、墨西哥城 |

**设计理念**：
- 可选维度，不选时默认为"通用场景"
- 与"建筑风格"叠加产生化学反应：
  - 广州 + 岭南建筑 = 传统西关大屋
  - 上海 + 岭南建筑 = 上海洋房里的岭南元素（混搭）
  - 东京 + 昭和复古 = 东京下町怀旧风

### 维度4：空间功能（输入+预设）

**定位**：这个空间用来做什么

**预设选项**（点击快速填充）：
- 客厅、卧室、书房、厨房、阳台
- 咖啡馆、书店、花店、 bakery
- 庭院、露台、天台
- 老街巷、市集、车站
- 茶室、画室、工作室

**输入框设计**：
```
[空间功能输入框]
placeholder: "输入空间功能，如'社区咖啡馆'、'阁楼画室'、'屋顶花园'..."

快捷标签：[客厅] [卧室] [咖啡馆] [书店] [庭院] [茶室] [画室]...
```

**设计理念**：
- 不再穷举，允许用户自由输入
- 提供快捷标签加速选择
- 支持复合空间描述（"社区咖啡馆角落的窗边座位"）

### 维度5：天气/氛围（单选）

**定位**：天气状况和特殊氛围

| 类别 | 选项 |
|-----|------|
| **晴朗** | 晴空万里、蓝天白云、阳光明媚 |
| **多云** | 云层覆盖、柔和散射、阴天静谧 |
| **雨天** | 细雨绵绵、倾盆大雨、雨后湿润 |
| **极端** | 暴雨天、台风过境、暴雪封门 |
| **特殊** | 雾天朦胧、沙尘漫天、极光之夜 |
| **黄昏/夜晚** | 金色黄昏、蓝调时刻、星光璀璨 |

**设计理念**：
- 简化为单选，避免与光影维度冲突
- 包含极端天气选项（暴雨、台风、暴雪）
- "黄昏/夜晚"作为特殊时段氛围

---

### 维度6：光影质感（多选）

**定位**：光线的质感特征，可多选叠加

| 选项 | 说明 |
|-----|------|
| **自然窗光** | 窗户透入的柔和自然光 |
| **侧逆光** | 侧面或背面打来的轮廓光 |
| **斑驳光影** | 树影、窗格等产生的光影图案 |
| **烛光/暖光** | 温暖的点光源氛围 |
| **雾气柔光** | 雾气中的散射柔光 |
| **霓虹反光** | 夜晚霓虹的彩色反光 |
| **电影感打光** | 类似电影布光的戏剧性光线 |

**设计理念**：
- 与"天气"解耦，只描述"光的质量"
- 可多选叠加（如：自然窗光 + 斑驳光影）
- 用通俗易懂的描述替代专业术语（去掉"黄金时刻"等）

---

### 维度7：绘画风格（多选）

**定位**：画面的绘制风格/质感

| 选项 | 说明 |
|-----|------|
| **精细漫画** | 插画感、线条精致、日系/国漫风 |
| **写实细节** | 精致细腻、照片级质感 |
| **胶片质感** | 模拟胶片色彩、颗粒感 |
| **水彩晕染** | 水彩画的晕染效果 |
| **油画质感** | 油画的笔触和厚重感 |
| **铅笔素描** | 手绘素描风格 |

**设计理念**：
- 将"年代维度"（80年代怀旧）移到这里的"胶片质感"
- 纯粹描述"绘画/渲染风格"

## UI展示顺序

```
1. 核心情绪*       （单选，突出显示）
2. 建筑风格*       （单选，accordion按地区分组）
3. 地域            （单选，可选，城市选择器）
4. 空间功能*       （输入框 + 快捷标签）
5. 天气/氛围*      （单选，图标化展示）
6. 光影质感        （多选checkbox）
7. 绘画风格        （多选checkbox）
8. 补充描述        （文本域）
```
